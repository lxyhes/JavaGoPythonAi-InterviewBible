数据结构

数组和链表的区别？
高频
特性 数组 链表
存储 连续内存 离散内存
访问 O(1)随机访问 O(n)顺序访问
插入/删除 O(n) O(1)
适用场景 频繁查询 频繁增删
栈和队列的区别？应用场景？

栈（Stack）：后进先出（LIFO）

  • 应用：函数调用栈、表达式求值、括号匹配、浏览器前进后退

队列（Queue）：先进先出（FIFO）

  • 应用：BFS、任务调度、消息队列、缓冲处理
二叉树的遍历方式有哪些？
高频

深度优先（DFS）：

  • 前序遍历：根-左-右
  • 中序遍历：左-根-右
  • 后序遍历：左-右-根

广度优先（BFS）：

  • 层序遍历：按层从上到下、从左到右
二叉搜索树、AVL树、红黑树的区别？
  • 二叉搜索树（BST）：左<根<右，最坏O(n)
  • AVL树：平衡二叉树，左右子树高度差≤1，查找O(logn)但旋转频繁
  • 红黑树：近似平衡，旋转较少，插入删除O(logn)，Java TreeMap/HashMap使用
哈希表的原理？如何解决冲突？
高频

原理：通过哈希函数将key映射到数组索引

冲突解决方法：

  • 链地址法：冲突元素组成链表（Java HashMap）
  • 开放寻址法：线性探测、二次探测、双重哈希
堆（优先队列）的特点？应用场景？

特点：完全二叉树，父节点大于/小于子节点

  • 大顶堆：父节点最大，用于升序排序
  • 小顶堆：父节点最小，用于降序排序

应用：Top K问题、合并K个有序数组、任务调度

时间复杂度：插入O(logn)，获取堆顶O(1)

算法思想

时间复杂度和空间复杂度如何分析？
重要

常见时间复杂度（从低到高）：

  • O(1)：常数时间
  • O(logn)：二分查找
  • O(n)：线性扫描
  • O(nlogn)：快速排序、归并排序
  • O(n²)：冒泡、选择、插入排序
  • O(2^n)：递归回溯

冒泡排序 O(n²) O(n²) O(1) 稳定
快速排序 O(nlogn) O(n²) O(logn) 不稳定
归并排序 O(nlogn) O(nlogn) O(n) 稳定
堆排序 O(nlogn) O(nlogn) O(1) 不稳定
动态规划的核心思想是什么？
重要

核心思想：将大问题分解为子问题，保存子问题的解避免重复计算

解题步骤：

  1. 定义状态（dp数组含义）
  2. 确定状态转移方程
  3. 初始化边界条件
  4. 确定遍历顺序

经典问题：斐波那契、爬楼梯、最长公共子序列、背包问题

贪心算法和动态规划的区别？
  • 贪心：局部最优，不回溯，不一定全局最优
  • 动态规划：保存所有子问题解，考虑各种情况，全局最优

贪心适用：活动选择、分数背包、哈夫曼编码

DP适用：0-1背包、最长递增子序列、编辑距离

回溯算法的框架是什么？
def backtrack(路径, 选择列表):
    if 满足结束条件:
        结果.add(路径)
        return
    
    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择  # 回溯

经典问题：全排列、N皇后、子集、组合总和

反转链表（LeetCode 206）
// 迭代法
ListNode reverseList(ListNode head) {
    ListNode prev = null, curr = head;
    while (curr != null) {
        ListNode next = curr.next;
        curr.next = prev;
        prev = curr;
        curr = next;
    }
    return prev;
}
两数之和（LeetCode 1）
// 哈希表 O(n)
int[] twoSum(int[] nums, int target) {
    Map map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int complement = target - nums[i];
        if (map.containsKey(complement)) {
            return new int[]{map.get(complement), i};
        }
        map.put(nums[i], i);
    }
    return null;
}
二叉树层序遍历（LeetCode 102）
List> levelOrder(TreeNode root) {
    List> result = new ArrayList<>();
    if (root == null) return result;
    
    Queue queue = new LinkedList<>();
    queue.offer(root);
    
    while (!queue.isEmpty()) {
        int size = queue.size();
        List level = new ArrayList<>();
        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
